### Backend

1. Run `cd server` on your CLI.

2. Install the dependencies by running
    ```
    nodemon to run development mode
    npm install
    ```
3. edit db.js line n0. 6 by adding your mySQL password.

4. Run the server
    ```
    npm run devStart
    ```


### Frontend

1. Run `cd client` on your CLI.

2. Install the dependencies by running
    ```
    npm install
    ```

3. Run the server
    ```
    npm start
    ```
